# Changelog

## [0.1.0] - 2024

### Добавлено
- Класс `Client` для удобной работы с API
- Автоматическая регистрация пользователя при создании клиента
- Валидация URL сервера
- Улучшенная обработка ошибок
- Метод `get_answer()` для получения только текста ответа
- Поддержка установки через pip
- Документация и примеры использования

### Улучшено
- Рефакторинг кода для лучшей читаемости
- Убраны print-выводы, заменены на возврат ошибок
- Добавлены таймауты для HTTP-запросов
- Улучшена типизация
- Оптимизированы зависимости (только requests)

### Исправлено
- Правильная обработка ошибок в request.py
- Корректное закрытие файлов в finally-блоках
